# Screenshots

These screenshots are generated from the local Mojo app for final A2 review evidence.

To refresh them:

```sh
npm run seed
npm run dev
```

In another terminal:

```sh
npm run screenshots
```

The script logs in as the seeded `Fanxin` member and captures:

- `home-desktop.png`
- `explore-desktop.png`
- `home-mobile.png`
- `log-mobile.png`

The screenshots are not required to run the prototype. They are included only as visual evidence for the final submission package.
